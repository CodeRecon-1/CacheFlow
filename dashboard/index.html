Drop-in Integration

Change only the base_url in your existing code. No other changes needed.

OpenAI (Python)
from openai import OpenAI # Before: client = OpenAI(api_key="sk-...") # After (one line change): client = OpenAI( api_key="sk-...", base_url="http://localhost:8000/openai" ) # All calls work identically: resp = client.chat.completions.create( model="gpt-4o", messages=[{"role": "user", "content": "Hello"}] )
Anthropic (Python)
import anthropic # Before: client = anthropic.Anthropic(api_key="sk-...") # After (one line change): client = anthropic.Anthropic( api_key="sk-...", base_url="http://localhost:8000/anthropic" ) # All calls work identically: msg = client.messages.create( model="claude-3-opus-20240229", messages=[{"role": "user", "content": "Hello"}], max_tokens=1024 )
Multi-Variant Generation

Add _optimizer_variants: N to generate N response variants in one batch call.

import httpx, json resp = httpx.post("http://localhost:8000/openai/v1/chat/completions", headers={"Authorization": "Bearer sk-..."}, json={ "model": "gpt-4o", "messages": [{"role": "user", "content": "Write a tagline for a coffee brand"}], "_optimizer_variants": 5 # ← magic field: generates 5 variants } ) # Returns: {"variants": [{"variant": 0, "text": "..."}, ...]} for v in resp.json()["variants"]: print(f"Variant {v['variant']}: {v['text'][:80]}")
